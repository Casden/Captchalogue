# Homestuck Asset Slots

This folder holds optional theme assets for the Captchalogue frontend.
The app remains functional without these files because CSS includes color fallbacks.

## Current file slots

- `logo/SBurb_Logo.svg`
  - Used in sidebar and connect-page branding.
  - Recommended: square SVG or transparent PNG fallback, at least 512x512.

- `cards/card-front.png`
  - Used on the front face of captchalogue cards.
  - Recommended: portrait card frame, transparent center.

- `cards/card-back.png`
  - Used on the reverse side of captchalogue cards.
  - Recommended: dark pattern, roughly 2:3 ratio.

- `designix/Punch_Designix.webp`
  - Used as the mint ceremony overlay backdrop.
  - Recommended: 1200px+ wide image.

- `animations/sburb_loading_animation.gif`
  - Displayed during active minting steps in the overlay.
  - Recommended: seamless loop, optimized for web (small file size).
